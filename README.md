# Sagnik Chakravarty — Personal Website

This repository contains the source code for my personal academic website, hosted via GitHub Pages.

---

## About

I am a graduate student in the Joint Program in Survey Methodology (JPSM) at the University of Maryland, College Park, specializing in Survey and Data Science.

My work focuses on the intersection of:
- Survey Methodology (sampling, measurement error, nonresponse)
- Computational Social Science
- Machine Learning and Data-Centric AI
- NLP and Large Language Models

This website serves as a centralized platform to present my research, projects, and academic work.
